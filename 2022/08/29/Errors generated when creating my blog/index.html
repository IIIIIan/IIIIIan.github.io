Errors generated when creating my blog

reference: https://blog.csdn.net/yushuaigee/article/details/111465155

1. errorno: -4048 or operation not permitted

When executing commands, Error: EPERM: operation not permitted reported.

Solution: Run cmd as Administrator and execute the command below again

2.Report deployer not found: git when running hexo g && hexo d

Solution: Download packet deployer-git using

##3. Report error when using hexo g && hexo d to update remote repository

This error is relatively rare and nobody has given the solution, I checked all the codes I have changed and finally found the bug.

I followed the instruction book by butterfly and set enable to false, then the error never occurs.

Solution:

Set enable to false

4. Error: Spawn failed

This error is due to wifi connection issues and is very common in China, here are two solutions.

Solution 1: Use a VPN and connect to nodes outside China
Solution 2: Connect to a cell phone cellular network
