Solve the problem of not being able to copy text from a web page

When you want to copy text form a web page, sometimes you may find that Ctrl+C and Ctrl+V doesn’t works, how to solve this problem?

Method:

Right Click and choose check to enter the console.

Type in $=0, enter

Reasons:

$=0 can prevent using Javascript, which is used to avoid copying.
