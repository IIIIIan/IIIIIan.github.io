Functions you can’t imagine using cmd

1. immediately auto power-off when clicked

Use Win+R to bring up the run box, type cmd and press Enter to use the cmd.

Type shutdown to see the parameters of shutdown

It’s clear that both /s and /p works, but /p works better here as no warning message will be shown.

Method:

Right-click on the desktop to create a new text document, name it automatic shutdown, and modify the file extension to bat

Right-click to edit, enter shutdown /p, press Ctrl+S to save

After returning to the desktop, click to force shutdown directly

Change the icon of the bat so others can’t know what it is

Right-click to create a shortcut, then right-click the created shortcut, select Properties

Click on the change icon

Also look at the icon of QQ

Copy the QQ icon address and paste it into the icon address of the bat shortcut, so that you will get a bat that is exactly the same as QQ

Then change the name of the bat to QQ, and delete the QQ shortcut from the desktop

2. Shutdown fancifully

The same procedure, the only difference is change the content into slidetoshutdown

It should be noted that system below win10 does not support this function

3. Crash code

The code can force the computer to keep pop-up cmd window and force the computer to shutdown

(omitted here)

The meaning of the code is to open cmd infinitely, this operation will quickly occupy the computer memory until it crashes, and the mouse and shortcut keys will not work for about two seconds after running

I haven’t found a way to cancel it halfway. It’s fine if it’s dead and restarted.

It can be used with the previous icon change operation, which will be a bit disgusting. .
